MH Wilds Build Checklist

An interactive build planner with dynamic DOM manipulation and localStorage.

HTML JavaScript CSS

An interactive build planner for Monster Hunter Wilds. Users can create builds, add gear to six equipment slots, check off items, and delete builds. All saved with localStorage.

Challenges: Getting localStorage to save and load correctly between sessions, and managing dynamic DOM elements for multiple builds.

What I learned: Arrays, objects, functions, DOM manipulation, event handling, and how to persist data with localStorage

Future plans: I want to replace the manual input fields with a search function connected to a database or API for Monster Hunter Wilds gear. This would let users search for weapons, armor, and decorations instead of typing them in manually. I'd also like to add decoration slots and automatic skill calculations to make it a complete build tool.
